# Ciclo de Conversas Design e Multimédia

This website was developed for the "Project 1 - Web Identity" course unit of the Bachelor's in Design and Multimedia of the Faculty of Sciences and Technology of the University of Coimbra. This project was made with Marta Babau, Rafa Arias and Silas Sequeira.

The main objective was to develop a website for Ciclo de Conversas Design e Multimédia, along with the visual identity for the event.

Ciclo de Conversas Design e Multimédia is an annual series of lectures organized by professors from the University of Coimbra.

## Library Used

**p5.js** <https://p5js.org/>
*p5.js is a friendly tool for learning to code and make art. It is a free and open-source JavaScript library built by an inclusive, nurturing community. p5.js welcomes artists, designers, beginners, educators, and anyone else!*
